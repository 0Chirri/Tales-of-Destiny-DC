# Tales of Destiny DC - Patch Instructions
Get xdeltaUI from https://www.romhacking.net/utilities/598/
1. Open xdeltaUI.exe
1. Select original ISO file
1. Select patch file .xdelta
1. Browse to a location to save the patched ISO file
1. Enjoy!
